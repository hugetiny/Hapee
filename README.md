Developing 0.1.0 ...



这个是我对未来搜索引擎的一个梦想，
尽量开源，尽量挖掘Dart在前端的潜藏能力，尽量支持所有协议，

为什么要起名Hapee？
因为我觉得人的一生最缺少快乐吧。希望Hapee能带给大家刚第一次接触到搜索引擎时轻易搜到自己喜欢的东西的那种快乐。

| Windows | x64, Arm64 | 10, 11 | 10 | 8 and earlier |
